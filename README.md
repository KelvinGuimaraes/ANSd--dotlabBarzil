# 🧬 ANSd - Dotlab Brazil

Projeto de website desenvolvido para apresentar informações sobre o **ANSd**, uma solução tecnológica voltada ao apoio no diagnóstico, monitoramento e acompanhamento da hanseníase no Brasil, com foco em tecnologias aplicadas à área da saúde.

O site foi desenvolvido para o **[Dotlab Brazil](https://dotlabbrazil.com.br/)** como uma **plataforma informativa e institucional**, apresentando os objetivos do projeto, seus desenvolvimentos e dados relevantes.

🌐 **Acesse o projeto online:**  
https://kelvinguimaraes.github.io/ANSd--dotlabBarzil/

---

# 📌 Sobre o Projeto

O **ANSd** é um projeto voltado ao desenvolvimento de soluções tecnológicas que auxiliam profissionais da saúde e **moderniza a Avaliação Neurológica Simplificada (ANS)**, utilizando ferramentas digitais para análise, organização e visualização de dados.

O site apresenta informações sobre:

- O projeto e seus objetivos
- Principais desenvolvimentos tecnológicos
- Dados e indicadores relacionados à área da saúde
- Estrutura do projeto e suas aplicações
